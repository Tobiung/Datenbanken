Datenbank Grundlagen

Was ist eine Datenbank?

Eine Datenbank ist eine systematische und zweckgerichtete Sammlung von Daten. Aus dem Verwendungszweck ergibt sich, welche Daten gesammelt und gespeichert werden.

Beispiele für Datenbanken sind:

  • Kundendatenbanken in Unternehmen
  • Produktdatenbanken in Online-Shops
  • Bibliothekskataloge
  • Kontaktverwaltung auf Smartphones

Was ist ein Datenbankmanagementsystem (DBMS)?

Ein Datenbankmanagementsystem (DBMS) ist die Software, mit der die Daten in einer Datenbank verwaltet werden können. Das DBMS stellt Methoden zur Verfügung, mit denen Nutzer auf die gespeicherten Daten zugreifen können.

Beispiele für DBMS:

  • MySQL
  • Oracle Database
  • Microsoft SQL Server
  • PostgreSQL
  • SQLite

Was ist ein Datenbanksystem (DBS)?

Ein Datenbanksystem besteht aus der Kombination einer Datenbank und eines Datenbankmanagementsystems. Es umfasst also sowohl die Daten selbst als auch die Software zu deren Verwaltung.

Zusammenhang:

Datenbanksystem (DBS) = Datenbank (DB) + Datenbankmanagementsystem (DBMS)
